Navigation Button

Rendered element:

Default Button

CSS selector:

".default-button"

Sample code:

<a href="#" class="default-button">Default Button</a>

In-text link

Rendered element:

link text

CSS selector:

"a" or ".links"

Sample code:

<a href="#" class="links">link text</a>

Other Button Stylings

Rendered element:

CSS selector:

".buttonstyle1"

Sample code:

<button class="buttonstyle1" type="button">Button Style</button>

Form Styling

Rendered element:

First name: Last name:

CSS selector:

".form-style"

Sample code:

<form action="#" class="form-style">
First name: <input type="text" name="fname">
<button class="buttonstyle1" type="submit">Submit</button>
</form>

Text Input Field

Rendered element:

Search nearby address:

CSS selector:

".input-style"

Sample code:

<div class="input-style">
Search nearby address: <input type="text" name="address">
</div>

Paragraphs

Rendered element:

body font uses Google font's Raleway typeface such as this text Headers and other text would use Lobster Two and Open Sans

CSS selector:

"p"

Sample code:

<p>paragraph content</p>

Bulleted List / Unordered List

Unordered lists are rendered without any list-style

Rendered element:

  • list item a
  • list item b

CSS selector:

"ul" and "li"

Sample code:

<ul>
<li>list item</li>
</ul>

Numbered List

Rendered element:

  1. list item a
  2. list item b

CSS selector:

"ol" and "li"

Sample code:

<ol>
<li>list item</li>
<li>list item</li>
</ol>

Grid Structure

Grid Description:

The grid currently being used on this site uses flexbox. The site is first divided into columns.
Then content are placed inside rows, which are further divided into maximum three columns.

CSS selector:

".row" for content sections and ".content-in-grid" for individual content

Sample code:

<p>paragraph content</p>
